# OfferPilot

「Offer 捕手」是面向学生求职场景的 AI 求职匹配智能体，目标是帮助学生从岗位信息中找到更匹配的机会，并针对目标岗位优化简历表达，提升简历初筛命中率。

## 第一阶段范围

第一阶段按网页端 MVP 设计：前端采用静态网页部署，可发布到 GitHub Pages；后端采用独立 API 服务承载简历解析、岗位解析、匹配评分、报告生成和 AI 编排能力。
